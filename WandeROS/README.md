# WandeROS
a Gazebo simulation of a wanderer navigation system for Turtlebot 3 Burger, implemented using ROS Melodic & rospy.

## Supported OS:
Ubuntu 18.04

## Usage:
```sh
python2 install_requirements.py  
python2 launch_simulation.py
```
